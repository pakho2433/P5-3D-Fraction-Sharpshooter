# P5 3D Fraction Sharpshooter

一個適合小五學生的 3D 分數加減射擊遊戲。

## 遊戲功能

- 左下虛擬搖桿控制角色行走
- 拖動畫面可 360° 自由轉動視角
- 桌面支援 WASD／方向鍵、滑鼠拖曳及空白鍵射擊
- 箭靶改成會移動的 3D 怪獸，答案以直式分數顯示在身上
- 每局最長 60 秒
- 射中正確答案後直接進入下一關，共 10 關
- 支援手機、iPad 及電腦瀏覽器
