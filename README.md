# P5 3D Fraction Sharpshooter

一個適合小五學生的 3D 分數加減射擊遊戲。

## 三個程度

- **程度一**：2 個分數加減；答案只是真分數；第 1–9 關每關 60 秒
- **程度二**：3 個分數加減；答案是帶分數；第 1–9 關每關 90 秒
- **程度三**：3 個分數加減；答案會輪流出現真分數、假分數及帶分數；第 1–9 關每關 90 秒
- 每次過關後，該程度的限時會自動重設
- 答案等於整數時只顯示整數，例如顯示 `1`，不顯示 `1/1`

## 遊戲功能

- 左下虛擬搖桿控制角色在擴大的森林場地行走
- 拖動畫面可 360° 自由轉動視角
- 桌面支援 WASD／方向鍵、滑鼠拖曳及空白鍵射擊
- 會移動的 3D 怪獸，答案以分數、假分數或帶分數顯示在身上
- 射錯答案一次扣 5 秒
- 箭會飛出並插在被射中的怪獸身上
- 第 6 關起怪獸會向玩家發射攻擊，被擊中一次扣 5 秒
- 第 10 關為不限時最終大怪獸戰
- 大怪獸 HP 為 10，會持續發射攻擊，HP 越低攻擊越快
- 射擊大怪獸本體不會扣 HP；每擊倒一隻正確答案的小怪獸，大怪獸 HP 才會減 1
- 大怪獸 HP 顯示移到畫面底部中央，不會遮擋題目
- 第 10 關被攻擊擊中時會被擊退及短暫不能射擊，但不會扣時間
- 支援手機、iPad 及電腦瀏覽器
